# Lab: Introducing Pi Through Stability

This lab introduces pi without beginning from decimal digits.

## Materials

- circular lids or wheels of different sizes
- string
- ruler
- marker
- paper

## Step 1: Radius pulse

Mark the center and radius of each circle.

Pulse:

```text
radius_seen = Y
```

## Step 2: Boundary pulse

Wrap string around the circle and mark one full boundary length.

Pulse:

```text
circumference_traced = Y
```

## Step 3: Compare

Compare the boundary string with the diameter.

The student should notice that the boundary is a little more than three diameters.

Pulse:

```text
boundary_to_diameter_relation_seen = Y
```

## Step 4: Stability

Repeat for different circles.

The key perception is that the relation stays stable even when the circle size changes.

Pulse:

```text
scaling_invariance_seen = Y
```

## Step 5: Symbol

Only now introduce:

```text
C = 2 × pi × r
```

The symbol is introduced after the perceptual invariant has appeared.
