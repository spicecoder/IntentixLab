# Lab Template: Geometry Rider as Pulse Field

This template can be used for any geometry rider.

## 1. Draw the situation

Place the diagram clearly. Avoid clutter. Mark only what is needed first.

## 2. Define visible Pulses

```text
line_AB_seen = Y
triangle_ABC_seen = Y
parallel_relation_seen = UN
angle_relation_seen = UN
```

## 3. Define theorem Pulses

```text
alternate_angles_available = UN
triangle_angle_sum_available = UN
congruence_test_available = UN
```

## 4. Ask perceptual questions

- What should be compared?
- What can move without changing the relation?
- Which mark looks irrelevant but may become important?
- Which theorem becomes visible if one more construction is added?

## 5. Stabilize one relation

Do not ask the learner to solve the whole rider immediately. Stabilize one missing Pulse.

## 6. Let proof emerge

Once enough Pulses become stable, the proof sequence should feel less arbitrary.
