# CPUX Tools

A collection of interactive tools to accelerate CPUX design, development, and analysis.

## Available Tools

### Entity Extractor
Extract intentions, pulses, and action contexts from scenario descriptions using AI. Supports both Anthropic Claude and OpenAI ChatGPT.

**Use cases:**
- Rapid prototyping of new CPUX systems
- Validating scenario completeness
- Discovering missing perceptions
- Learning CPUX design patterns
