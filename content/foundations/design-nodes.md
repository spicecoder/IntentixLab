# Design Nodes

A Design Node (DN) is a black-box Signal transformer.

```text
perform(inputSignal) -> outputSignal
```

A DN:
- absorbs Signals,
- transforms contextual meaning,
- emits Signals.

A DN is also refered as a Determinitic intention function within a CPUX Intention Container.
